# Spotimetrics
Web-based tool for personal Spotify statistics using Spotify Web API.

## Website
Visit the tool here: [https://spotimetrics.web.app](https://spotimetrics.web.app)

## Description
A web tool built with **HTML, CSS, and Vanilla JavaScript** to query and visualize your most played tracks and artists from Spotify.

## Features
- **Top Statistics:** View your most played Tracks and Artists.
- **Time Ranges:** Toggle between Short Term, Medium Term, and Long Term data.
- **Secure Authentication:** Implements OAuth 2.0 with PKCE flow for secure data access.
- **Responsive UI:** Fully centered grid layout with a dark-themed Spotify aesthetic.

## Tech Stack
- **Languages:** HTML5, CSS3, JavaScript (ES6+)
- **API:** [Spotify Web API](https://developer.spotify.com/documentation/web-api/)
- **Authentication:** OAuth 2.0 + PKCE Flow
- **Hosting:** Firebase
