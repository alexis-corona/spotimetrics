# Changelog

## [1.0.0] - 2026-04-02
### Added
- **Top Tracks View:** Display your most played songs across 4 weeks, 6 months, and All Time.
- **Top Artists View:** Show your favorite artists with high-quality profile images.
- **View Toggle:** Seamless switching between Tracks and Artists statistics.
- **Improved UI:** Fully centered grid layout with a dark-themed Spotify aesthetic.
- **Spotify Web API Integration:** Secure authentication using OAuth 2.0 with PKCE flow for personal data access.
- **GitHub Link:** Quick access to the source code directly from the navbar.
